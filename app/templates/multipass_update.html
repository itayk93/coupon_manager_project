עדכון קופונים מ-Multipass (GitHub)

עמוד זה מיועד להרצת תהליך העדכון דרך GitHub Actions (Playwright).

התהליך מתבצע ברקע ועשוי לקחת מספר דקות.

Cron Job Setup (User ID 1)

To schedule an automatic update, trigger the following API endpoint:

Endpoint: POST {{ url_for('coupons.api_update_multipass', _external=True) }}
Headers: X-Cron-Secret: <YOUR_API_TOKEN>
Mode: default dispatches GitHub Actions only (fast). Use ?mode=full to run the full server-side flow.
# Example CURL command:
curl -X POST "{{ url_for('coupons.api_update_multipass', _external=True) }}" \
  -H "X-Cron-Secret: {{ config.get('CRON_API_TOKEN', 'YOUR_TOKEN') }}"
# Full server-side mode (optional):
curl -X POST "{{ url_for('coupons.api_update_multipass', _external=True) }}?mode=full" \
  -H "X-Cron-Secret: {{ config.get('CRON_API_TOKEN', 'YOUR_TOKEN') }}"
# Crontab Example (Run every hour):
0 * * * * curl -X POST "{{ url_for('coupons.api_update_multipass', _external=True) }}" -H "X-Cron-Secret: {{ config.get('CRON_API_TOKEN', 'YOUR_TOKEN') }}" >> /tmp/cron_log.txt 2>&1

Note: Ensure CRON_API_TOKEN is set in your environment variables. The example above uses your current CRON_API_TOKEN.
